Heart Rate Sensor & Formula 1 Safety Gear Testing

Arduino MATLAB Prototyping

A real-time heart rate monitor built to analyze biometric data under simulated Formula 1 racing conditions.

Overview

  • Built the sensor using MAX30102 pulse oximetry and an Arduino with an OLED display.
  • Analyzed and visualized BPM data in MATLAB across simulated Formula 1 safety gear conditions.
  • Presented findings at an engineering showcase.

Process

1

Hardware build

Wired the MAX30102 to an Arduino with an OLED readout for live BPM display.

2

Data collection

Logged heart-rate data under simulated F1 safety gear conditions to study thermal/physiological load.

3

Analysis in MATLAB

Processed and visualized BPM trends across test conditions.

4

Showcase

Presented the results at an engineering showcase.
